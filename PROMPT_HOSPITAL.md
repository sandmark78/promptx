# 🚑 坏提示急救室 - 提示词纠错系统

**定位**: 解决用户真实痛点 - "龙虾老是乱答/胡说八道"

---

## 📋 常见坏提示库

### 1. 太短/太模糊
```
❌ 坏提示："帮我写个文案"
✅ 好提示：
<instructions>
你是一位专业的小红书文案策划。
请为 [产品名] 创作一篇种草文案。
</instructions>
<context>
目标受众：25-35 岁女性
产品亮点：天然成分、性价比高
</context>
<format>
- 标题：包含 emoji，15 字以内
- 正文：300-500 字，分 3-4 段
- 标签：5-8 个相关 hashtag
</format>
```

**问题诊断**: 缺少角色、场景、格式
**改进建议**: 添加 `<instructions>`, `<context>`, `<format>`

---

### 2. 没有角色定义
```
❌ 坏提示："解释一下量子力学"
✅ 好提示：
<instructions>
你是一位物理学教授，擅长用通俗语言解释复杂概念。
请用高中生能理解的方式解释量子力学。
</instructions>
```

**问题诊断**: 缺少角色定位
**改进建议**: 添加角色定义，明确受众水平

---

### 3. 没有输出格式
```
❌ 坏提示："分析这个产品的优缺点"
✅ 好提示：
<instructions>
请分析 [产品名] 的优缺点。
</instructions>
<format>
使用表格格式：
| 优点 | 缺点 |
|------|------|
| ... | ... |

最后给出购买建议（100 字以内）
</format>
```

**问题诊断**: 输出格式不明确
**改进建议**: 用 `<format>` 指定表格/列表/段落

---

### 4. 逻辑冲突
```
❌ 坏提示："写一篇 500 字的文章，要非常详细"
✅ 好提示："写一篇 500 字的文章，涵盖以下 3 个要点..."
```

**问题诊断**: 字数与详细度冲突
**改进建议**: 明确优先级或调整字数

---

### 5. 否定词用错
```
❌ 坏提示："不要写太长，不要用专业术语，不要废话"
✅ 好提示："保持简洁（300 字以内），用通俗语言，直接给出核心观点"
```

**问题诊断**: 否定词过多，模型容易忽略
**改进建议**: 用肯定句表达期望

---

### 6. 权重滥用
```
❌ 坏提示："非常重要！！！一定要！！！必须！！！"
✅ 好提示："请特别注意以下几点：1... 2... 3..."
```

**问题诊断**: 情绪化表达无效
**改进建议**: 用结构化列表强调重点

## 🏥 提示词体检功能

### 体检维度

| 维度 | 权重 | 检查项 |
|------|------|--------|
| 角色定义 | 20% | 是否有明确角色？ |
| 任务描述 | 25% | 任务是否清晰？ |
| 上下文 | 15% | 是否有背景信息？ |
| 输出格式 | 20% | 是否指定格式？ |
| 示例 | 10% | 是否有 few-shot？ |
| 边界条件 | 10% | 是否有禁止项？ |

### 评分标准

```
90-100: 完美提示词 🏆
80-89: 优秀，微调即可 ✨
70-79: 良好，有改进空间 👍
60-69: 及格，建议优化 ⚠️
<60: 需要重写 🚑
```

## 🚑 急救流程

```
1. 用户粘贴"坏提示"
   ↓
2. 自动诊断问题 (AI 分析)
   ↓
3. 生成体检报告 (得分 + 问题列表)
   ↓
4. 提供改进版本
   ↓
5. 解释为什么这样改有效
```
