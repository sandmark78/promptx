# 🎨 PromptCraft

> **OpenClaw 提示词扩展工具** - 将简单想法扩展为专业级提示词

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/sandmark78/openclaw-prompt-craft)

---

## ✨ 功能特性

- 🚀 **一键扩展** - 输入简单想法，生成专业提示词
- 📚 **50+ 模板** - 覆盖内容创作、代码开发、数据分析、商业应用
- 📜 **历史记录** - 本地保存最近 20 条使用记录
- 📋 **一键复制** - 快速复制到剪贴板
- 🎨 **精美界面** - 响应式设计，支持移动端
- 🔒 **完全免费** - 开源免费，无隐藏费用

## 🎯 使用示例

### 输入
```
帮我写一个数据分析报告
```

### 输出
```xml
<instructions>
你是一个数据分析专家，专门帮助企业管理层理解业务趋势。
你的任务是分析提供的数据并生成专业报告。
</instructions>

<context>
报告将被 CEO 和董事会阅读，需要专业、简洁、数据驱动。
避免技术术语，用商业语言表达洞察。
</context>

<format>
使用 <executive_summary>、<key_metrics>、<insights>、<recommendations> 标签结构化输出。
</format>

<examples>
...3-5 个高质量示例...
</examples>
```

## 🚀 快速开始

### 在线使用
访问：[https://openclaw-prompt-craft.vercel.app](https://openclaw-prompt-craft.vercel.app)

### 本地部署
```bash
# 克隆仓库
git clone https://github.com/sandmark78/openclaw-prompt-craft.git

# 进入目录
cd openclaw-prompt-craft

# 用浏览器打开 index.html 即可
open index.html
```

### Vercel 部署
1. Fork 本仓库
2. 在 Vercel 导入项目
3. 自动部署完成
